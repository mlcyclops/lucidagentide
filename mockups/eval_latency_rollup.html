Method & sources

Latency = time-to-first-token (TTFT), captured per model call at the request seam (acp_backend) and stored in api_latency; rolled up per model x hour over the period. Buckets use America/New_York so EST/EDT DST is handled. p50/p95 are percentiles over calls in each hour. Total-latency and error-rate rollups available on the same table.
